What Can JavaScript Do?

JavaScript can change HTML content.

Open your developer console:
  • Windows
    • Firefox: Crtl+Shift+K
    • Chrome: Crtl+Shift+J
    • Edge: F12
  • Mac
    • Firefox: Crtl+Shift+I
    • Chrome: Crtl+Option+J
    • Safari:
      1. Enable Develop menu in Preferences, (Command + ,) -- Advanced -- Show Develop menu in menu bar
      2. Alt + Command + C
  • Linux
    • Firefox: Crtl+Shift+I
    • Chrome / Chromium: Crtl+Option+J
